1. Variables: Las variables son almacenadores donde podremos guardar casi cualquier cosa que necesitemos más adelante (Sea texto, números y demás).
2. Strings: String es la manera en que llamamos a los textos en nuestro código. Esto no es solo palabras, sino que también es TODO lo que se encuentre entre comillas.
3. Funciones: Las funciones son muy parecidas a las variables. Pero, en vez de guardar texto o números (es decir, cosas pequeñas), aquí podremos almacenar código (el cual haga algo) y lo llamaremos cuando necesitemos. En las funciones existe algo que se llama 'argumentos', estos son datos tipo X (Es decir, datos que todavía o no tenemos o desconocemos por el momento). Esto nos servirá para poder ingresarlo al llamar la función, pudiendo tener muchas veces el mismo código PERO con distintos datos. Por último, tenemos lo que se llama 'return'. Esta es la mejor manera para devolver el resultado de una función, este resultado puede ser un texto, número o, inclusive, una variable (Es decir, el resultado de la función).
4. Declaraciones if: Esta declaración es como decir 'Si pasa esto, hago esto'. Es una declaración que siempre va a devolver un verdadero (Pasó lo que dije que iba a pasar) o falso (No pasó lo que dije que iba a pasar).
5. Valores booleanos: Existen 2 valores booleanos, el 'true'/verdadero y el 'false'/falso. Estas son 2 posibles respuestas que podríamos conseguir de una declaración.
